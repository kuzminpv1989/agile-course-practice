## Задание

Реализовать класс кватернионов с поддержкой основных операций:

 - Сложение
 - Вычитание
 - Умножение
 - Деление
 - Получение модуля
 - Получение сопряженного кватерниона

Все необходимые математические сведения можно найти в статье
["Кватернионы"][quaternion] на сайте Wikipedia.
